# Twitter API Integration

This project demonstrates how to interact with the Twitter API (now X API) to authenticate and post tweets using Python. It uses the `requests` library for API calls and `python-dotenv` to manage environment variables securely.

## Features
- **Authentication**: Authenticate with the Twitter API using OAuth 1.0a.
- **Post Tweets**: Post tweets programmatically using the Twitter API v2.
- **Environment Variables**: Securely store and manage API keys and tokens using a `.env` file.

## Prerequisites
Before running the project, ensure you have the following:
1. A [Twitter Developer Account](https://developer.twitter.com/).
2. A Twitter Developer App with **Read and Write** permissions.
3. Python 3.7 or higher installed.
